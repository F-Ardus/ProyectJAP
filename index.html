Competencias Transversales

Las competencias transversales son aquellas competencias que son comunes a cualquier profesión.
Estas son: Competencias Técnicas (Saber), Competencias Metodológicas (Saber Hacer), Competencias Participativas (Saber Estar) y Competencias Personales (Saber Ser).

Competencias Técnicas: Es cuando se aplican todos los conocimientos y habilidades.

Competencias Metodológicas: Es el saber aplicar los conocimientos y habilidades necesarios en cada situación laboral.

Competencias Participativas: Son aquellas que permiten a una persona relacionarse y comportarse en un ambiente de equipo.

Competencias Personales: Son todos los conocimientos adquiridos en nuestra vida que nos permiten trabajar de forma organizada y autonoma.

HTML

Como lo describe su nombre (Hypertext Markup Language) HTML es un lenguaje de marcado mediante etiquetas para la creacion de paginas web. Este es usado para definir la estructura de la misma pudiendo aplicar elementos como texto, fotos, videos, etc. Tambien contiene algunas propiedades para darle estilo a la pagina web, pero generalmente se utilizan también otros lenguajes para complementarlo tales como CSS (Cascading Stylesheets) para esilos y JS (JavaScript) para agregar interfaz e interacción a el sitio web.

CSS

Como mencionamos anteriormente CSS o Cascading Stylesheets es un lenguaje utilizado para dar estilos y diseño a una pagina web. Debido a que este se escribe en un archivo separado al HTML facilita la escritura y lectura de ambos. Vale tener en cuenta que normalmente tambien se agregan estilos mediante HTML ya que tienen precedencia sobre el archivo CSS, como asi tambien hay diferentes tipos de presedencia dentro de el mismo archivo CSS para facilitar la escritura y orden del mismo.

JavaScript

A diferencia de HTML y CSS que sirven para darle estructura y diseño al sitio web, JS sirve para hacer el mismo mas interactivo y dinamico. Este lenguaje permite el uso de operaciones lógicas que ademas de la web pueden ser utilizadas para aplicaciones de escritorio entre otros.

Jquery

Jquery es una biblioteca de Javascript que es utilizada para simplificar el uso de este, desde su interaccion con el código HTML, el uso de AJAX y otras tecnicas de desarrollo web.

Bootstrap

Por último definiremos a Bootstrap como un framework o un conjunto de herramientas para el diseño de paginas web. El mismo provee al desarrollador con archivos de CSS y JS personalizados para una mejor organización del sitio, asi como tambien proporciona varios bloques de codigo para agregar diversos elementos al mismo (barras de navegacion, botones, metodos de estructurado, etc).
